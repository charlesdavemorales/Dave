Electronic Locker System with Secure Access Control

Third Year Project - Embedded Systems

Project Overview

A microcontroller-based electronic locker system featuring secure password authentication, user management, and activity logging. The system provides a reliable and secure solution for personal storage access control.

Key Features

  • Keypad-based password entry
  • LCD display for user feedback
  • Electronic lock mechanism control
  • Master password override
  • Failed attempt tracking
  • Visual and audio feedback

Technologies Used

Arduino C++ Embedded Systems Security

System Architecture

The locker system is built around an Arduino microcontroller that interfaces with a 4x4 matrix keypad for password input, a 16x2 LCD for user interface, and a servo motor or solenoid lock for physical access control. The system includes LED indicators and a buzzer for visual and audio feedback.

Password data is stored securely in EEPROM memory, allowing persistence across power cycles. The system implements a timeout mechanism for failed login attempts and provides administrative functions for password reset and system configuration.

Security Features

  • Encrypted password storage
  • Lockout after multiple failed attempts
  • Master key access for emergencies
  • Activity logging with timestamps

Learning Outcomes

This project provided hands-on experience with embedded systems development, real-time programming, and hardware interfacing. It enhanced my understanding of security implementations in embedded applications and taught me about user experience design in resource-constrained environments.
